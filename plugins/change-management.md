# change-management
[NetBox Labs change-management docs](https://netboxlabs.com/docs/developer/plugins-extensions/changes/)
* Extends branching
* Collaborate with team members
* Define policies - determines who is eligible to approve a change request and how many approvals are required, contains one or more rules. Are selected by branch user when doing a change request.
* Request reviews, raise changes - based on defined policies and processes
* Designated change reviewers inspect change details - Can add comments (including on specific changes within the branch, citing any concerns or suggestions also - can be replied to by other users) or request updates, add review, approve/apply a change
* Once approved, branch can be merged into main branch  (once successfully merged, change request's status updates to 'completed')
* Easily revert changes
