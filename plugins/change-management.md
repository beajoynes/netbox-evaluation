# [change-management](https://netboxlabs.com/docs/developer/plugins-extensions/changes/)
* Extends branching
* Collaborate with team members
* Define policies
* Request reviews, raising changes - based on defined policies and processes
* Designated change reviewers - Can add comments or request updates, add review, approve/apply a change
* Once approved, branch can be merged into main
